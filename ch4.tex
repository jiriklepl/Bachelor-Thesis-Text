\section{Example: Polymorphic lists}

Polymorphic lists can be implemented as follows:

<a : b ~ struct List : <a> >
struct List
{
    a value;
    b *next;
};

Then we can implement its methods as follows:

<a : b ~ struct List : <a> >
void push(b *head, a value)
{
    b *item = new(sizeof(b));
}
